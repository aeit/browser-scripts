Проблема: 
Информация о пине частично скрыта в HTML из-за региональных или внутренних политик Pinterest (например, авторские права), что делает её недоступной через обычный интерфейс.

Решение: 
Скрипт извлекает данные пина из JSON, встроенного в страницу, и добавляет две кнопки:
•	B (Board) — копирует название и URL доски, открывает доску в новой вкладке;
•	A (Author/Pinner) — копирует имя и URL автора, открывает профиль в новой вкладке.
Если данные отсутствуют, показывается временное уведомление ("Доска не найдена" / "Автор не найден"). Кнопки поддерживают клавиатурные сокращения (A/B).

Особенности:
•	Используется безопасное извлечение данных с проверкой наличия полей;
•	Обработка ошибок при парсинге JSON;
•	Визуальная обратная связь: кнопки могут менять цвет при успешном извлечении данных;
•	Простая возможность расширения функционала.

Потенциальные улучшения:
•	Перемещать кнопки прямо над пином для более интуитивного взаимодействия;
•	Добавлять новые кнопки для дополнительных действий (например, копирование изображения, описание пина);
•	Использовать визуальные индикаторы состояния (цвет кнопки, иконка) для мгновенной обратной связи;
•	Настраиваемое позиционирование и стиль кнопок для разных разрешений экрана или мобильных устройств;
•	Возможность временно копировать URL при наведении, не дожидаясь клика.
